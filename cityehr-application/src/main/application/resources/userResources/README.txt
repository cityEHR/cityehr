	The userResources folder contains resources that may be useful for using cityEHR.
	These include template and sample files for various activities, including:

		the information model for the default cityEHR application
		template spreadsheets, mind maps and graph files for creating new models
		a sample patient record, Timothy Abernathy, for use with the default cityEHR application
		a set of sample users
		a sample entry directory, GPData, for use with the default cityEHR application
